# SmartCart-E-Commerce-Clustering-System

I built SmartCart — an e-commerce customer segmentation system that does something really interesting. Instead of telling the machine what to look for, I let it discover hidden customer groups on its own using unsupervised machine learning.
Working with 2,240 real customer records, I engineered features like total spending, customer tenure, and family size, then used PCA to compress the data and Agglomerative Clustering to identify 4 distinct customer personas — from budget-conscious family shoppers to high-spending premium buyers. Each cluster came with its own marketing story, turning raw data into actual business decisions.

Tech Stack — Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
ML Concepts — Unsupervised Learning, KMeans, Agglomerative Clustering, PCA, Elbow Method, Silhouette Score, Feature Engineering
Business Impact — Enabled data-driven personalized marketing strategies replacing generic one-size-fits-all campaigns, directly improving customer retention and marketing ROI.
